GET /v1/health - Health check for all services

Description: Monitor the health status of all microservices

Response Example: 
{
  "gateway_status": "healthy",
  "services": {
    "yolo": {"status": "healthy", "response_time": 0.123},
    "resnet": {"status": "healthy", "response_time": 0.089}
  }
}
POST /v1/chat/completions - OpenAI chat completions

Description: Generate chat responses using OpenAI models

Request Example: 
{
  "model": "gpt-4",
  "messages": [{"role": "user", "content": "Hello!"}],
  "temperature": 0.7,
  "max_tokens": 150
}
POST /api/mask/face - Face detection for masking

Description: Detect faces in images and return bounding boxes

Input: Multipart form with image file

Response Example: 
{
  "data": [
    [100.5, 150.2, 200.8, 250.9],
    [300.1, 400.3, 450.7, 550.2]
  ]
}
POST /api/mask/location - Location content detection

Description: Detect location-related content and return bounding boxes

Input: Multipart form with image file

Response Example: 
{
  "data": [
    [50.0, 75.5, 180.3, 120.8],
    [250.2, 300.1, 400.6, 380.9]
  ]
}
POST /api/mask/pii - PII detection in OCR data

Description: Detect personally identifiable information and return bounding boxes

Input: Multipart form with image file + OCR values JSON

OCR Input Example: 
[{
  "text": "john.doe@email.com",
  "bbox": [100, 200, 300, 220],
  "confidence": 0.95
}]
Response Example: 
{
  "data": [
    [120.0, 200.5, 350.8, 230.2],
    [400.1, 450.3, 600.7, 480.9]
  ]
}
